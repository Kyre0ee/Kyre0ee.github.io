### 方法1：字符串类型的变量可以使用运算符+进行连接
value = "hhh"
print("my test" + value + "ok")
### 方法2：使用逗号分隔变量和字符串
print("hello",value)
### 方法3：使用字符串格式化
print("hello {} ok".format(value))
### 方法4：使用f-strings打印变量和字符串
f-strings是一种更好、更易读、更简洁的实现字符串格式化方法
print(f"hello {value} ok")
